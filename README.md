# Prototype 1 – Data Reading App (Python)
This application displays live data received via the Arduino’s serial port.
Since this is an early prototype, the code is intentionally simple and focuses on core functionality:

- reading serial data from the Arduino
- updating a customtkinter GUI in real time
- displaying the incoming values in a clean interface

The goal of this prototype is to verify stable serial communication and real‑time data visualization before integrating more complex features.
